# 快速使用指南

## 🚀 三步开始使用

### 第一步：安装依赖（首次使用）

```bash
cd ~/桌面/pdf_zhongjiban
pip3 install -r requirements.txt
```

或者直接双击运行 `启动GUI.sh`，脚本会自动检查并安装依赖。

### 第二步：启动程序

**方式一：双击启动脚本**
```
双击 "启动GUI.sh"
```

**方式二：命令行启动**
```bash
cd ~/桌面/pdf_zhongjiban
python3 -m src.main --gui
```

### 第三步：处理PDF

1. **导入文件**（两种方式）：
   - 点击"浏览..."按钮选择PDF
   - 直接拖拽PDF文件到输入框

2. **调整参数**（可选）：
   - 字符缩放：默认1.5
   - DPI：默认300
   - 字间距：默认10
   - 行间距：默认20

3. **开始处理**：
   - 点击"开始处理"按钮
   - 等待处理完成

## 🎯 核心功能

### 自动方向检测

工具会**逐页检测**PDF的方向：

- **横版页面** → 使用横版算法（从左到右，从上到下）
- **竖版页面** → 使用竖版算法（从右到左，从上到下）
- **混合PDF** → 每页独立检测，自动选择算法

### 拖拽导入

支持的文件格式：
- `.pdf` - PDF文件
- `.jpg` / `.jpeg` - JPEG图片
- `.png` - PNG图片

直接拖拽到"输入文件"输入框即可。

## ⚙️ 参数说明

### 字符缩放 (scale_factor)

- **默认值**: 1.5
- **作用**: 控制输出字符的大小
- **建议**: 1.0 - 2.0 之间

### DPI

- **默认值**: 300
- **作用**: 控制输入和输出的分辨率
- **建议**:
  - 高质量：300-600
  - 标准质量：200-300
  - 快速处理：150-200

### 字间距 (char_spacing)

- **默认值**: 10 像素
- **作用**: 控制字符之间的间距
- **建议**: 5 - 20 之间

### 行间距 (line_spacing)

- **默认值**: 20 像素
- **作用**: 控制行与行之间的间距
- **建议**: 10 - 30 之间

## 🐛 常见问题

### Q: 拖拽功能不可用？

**A**: 需要安装 `tkinterdnd2`：
```bash
pip3 install tkinterdnd2
```

如果仍不可用，使用"浏览..."按钮选择文件。

### Q: 方向检测不准确？

**A**:
1. 确认PDF是扫描版（不是文字版）
2. 如果持续误判，可以在代码中强制指定方向（见README.md）

### Q: 字符识别不完整？

**A**:
- 可能是形态学参数不适合当前PDF
- 建议先备份，再谨慎调整参数（见README.md）

### Q: 处理速度慢？

**A**:
- 降低DPI可以加快速度（但会影响质量）
- 大文件需要耐心等待
